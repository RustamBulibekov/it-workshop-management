## 4. Relación entre Issues y Milestones

**Investigación realizada:** [Fecha]

### ¿Cómo se relacionan Issues y Milestones en GitHub?

1. **Estructura jerárquica:**
   - Un **Milestone** representa un objetivo o fase del proyecto (ej: "Sprint 1")
   - Las **Issues** son tareas específicas dentro de ese objetivo

2. **Relaciones principales:**
   - Varias Issues pueden pertenecer a un Milestone
   - El Milestone muestra progreso (porcentaje) basado en Issues completadas
   - Las Issues heredan la fecha límite del Milestone

3. **Beneficios de esta relación:**
   - **Agrupación lógica:** Issues relacionadas se juntan bajo un Milestone
   - **Seguimiento:** Ver qué tareas componen cada fase del proyecto
   - **Planificación:** Asignar fechas a grupos de trabajo
   - **Visualización:** Milestones muestran roadmap del proyecto
